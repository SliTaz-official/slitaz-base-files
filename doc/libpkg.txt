
SYNOPSIS
	. /usr/lib/slitaz/libpkg.sh
	function

DESCRIPTION
	Libpkg provides base functions to handle SliTaz Packages archives and
	receipts. It is used by Spk, the Cookutils and all tools dealing with
	packages and receipts.
